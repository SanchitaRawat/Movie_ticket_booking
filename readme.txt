Movie ticket booking system is designed simply to manage and keep track of ticket reservation, using technologies like HTML/CSS/Bootstrap for frontend. The backend is built using PHP that facilitates database management.

Steps to run in local system

1. Open PHPMyAdmin (http://localhost/phpmyadmin)

2. Create database with name theater_db

3. Import theater_db.sql file

4.Run the script http://localhost/Movie_Seat_Reservation_System
